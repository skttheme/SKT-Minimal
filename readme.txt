== Description ==

SKT Minimal is a minimalistic theme for neat white website design and clean user interface. Suitable for people who want to have a slate type website. Material and flat design UI for interior design, architecture, photography, creative, portfolio, software, consulting, furniture, digital and infographics type of site. Elementor based and comes packed with over 120+ other ready to use templates. Fast, easy and simple to use.
